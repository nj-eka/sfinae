for integral ip output to out ( sep )-separated byte-by-byte repr of integral type ip.

for string ip output to out ip element by element with sep - separeted.

for container ip output to out ip element by element with sep - separated.

for string ip output to out ip as is.

If address is integer than it's output byte-by-byte, unsigned, high byte first, followed by . as a separator.

Examples:
print_ip( int8_t{-1} ); // 255
print_ip( int16_t{0} ); // 0.0
print_ip( int32_t{2130706433} ); // 127.0.0.1
print_ip( int64_t{8875824491850138409} );// 123.45.67.89.101.112.131.41
